# Работа с каналами

## 1. Присоединение к каналу

### Opcode: `57`

#### Пример запроса

```json
{
  "ver": 11,
  "cmd": 0,
  "seq": 1,
  "opcode": 57,
  "payload": {
    "link" "https://max.ru/smi2"
  }
}
```

| Аргумент   | Тип  | Обязательный | Описание |
|:-----------|:----:|:------------:|:---------|
| `link`     | str  | ✅           | Ссылка на канал|
