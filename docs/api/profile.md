# Действия с профилем

## Изменение имени

### Opcode - `16`

#### Пример запроса

```json
{
  "ver": 11,
  "cmd": 0,
  "seq": 20,
  "opcode": 16,
  "payload": {
    "firstName": "user1",
    "lastName": "",
    "description": ""
  }
}
```

| Аргумент                      | Тип | Обязательный | Описание                              |
| :---------------------------- | :---:| :-----------: | :------------------------------------ |
| `firstName` | str | ✅ | Имя |
| `lastName` | str | ❌ | Фамилия |
| `description` | str | ❌ | Описание профиля |
